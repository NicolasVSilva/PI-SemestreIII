Treinamento base

Puxada aberta na polia

A puxada aberta na polia é um excelente exercício para trabalhar a parte superior das costas, com foco nos músculos dorsais, especialmente o latíssimo do dorso. Realizado com uma barra na polia alta, a puxada aberta envolve segurar a barra com as mãos em uma pegada mais afastada que a largura dos ombros, e puxá-la em direção ao peito.

Remada Curvada

A remada curvada é um exercício fundamental para o desenvolvimento das costas, especialmente para trabalhar os músculos dorsais, romboides, trapézio e lombares. Realizado com barra ou halteres, o movimento é feito a partir de uma posição curvada, com os joelhos ligeiramente flexionados e o tronco inclinado para frente, mantendo a coluna alinhada.

Remada martelo no banco inclinado

A remada martelo com halteres no banco inclinado é um excelente exercício para trabalhar as costas, especialmente os músculos dorsais, romboides e trapézio, além de envolver os bíceps de forma significativa.

Remada serrote no banco

A remada serrote no banco é um exercício eficaz para o desenvolvimento das costas, focando principalmente nos músculos dorsais, romboides e trapézio, além de trabalhar o bíceps e melhorar a estabilidade do core.

Pulldown

O pulldown é um exercício clássico para trabalhar a parte superior das costas, especialmente o latíssimo do dorso, além de ativar os bíceps e os músculos do trapézio.
